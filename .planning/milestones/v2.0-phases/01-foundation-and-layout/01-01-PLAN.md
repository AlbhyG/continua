Create `src/app/globals.css` with the following content:

```css
@import "tailwindcss";

@theme {
  --color-foreground: rgb(7, 7, 8);
  --color-muted: rgb(7, 7, 8);
  --color-accent: rgba(67, 117, 237, 0.92);
  --color-accent-light: rgba(169, 137, 236, 0.92);
  --color-border: rgba(255, 255, 255, 0.3);
  --color-card: rgba(255, 255, 255, 0.77);
  --color-card-solid: #ffffff;
}

html {
  scroll-padding-top: 80px;
}

body {
  font-size: 16px;
  line-height: 1.6em;
  letter-spacing: -0.02em;
  font-weight: 400;
  color: var(--color-foreground);
  background: linear-gradient(
    180deg,
    rgba(67, 117, 237, 0.92) 0%,
    rgba(169, 137, 236, 0.92) 35.14%,
    rgb(229, 158, 221) 100%
  );
  background-attachment: fixed;
  min-height: 100vh;
}

p + p {
  margin-top: 20px;
}
```

Key details:
- `@import "tailwindcss"` must be the first line (Tailwind v4 entry point).
- `@theme` block defines all style guide color tokens. These auto-generate utility classes: `bg-accent`, `text-foreground`, `bg-card`, `border-border`, etc.
- `--color-card: rgba(255, 255, 255, 0.77)` provides the glassmorphic card background. At worst-case gradient position (pink rgb(229,158,221)), the blended color behind text is approximately rgb(224,161,222) blended with 77% white = approximately rgb(241,221,240). Foreground text rgb(7,7,8) against this yields a contrast ratio well above 4.5:1 (approximately 16:1), satisfying WCAG 2.1 AA. If testing reveals issues, increase card opacity.
- `background-attachment: fixed` keeps gradient stationary on scroll per FOUN-01.
- Paragraph spacing `p + p { margin-top: 20px; }` per style guide.

Update `src/app/layout.tsx` to:

```typescript
import type { Metadata } from "next";
import { Inter } from "next/font/google";
import "./globals.css";

const inter = Inter({
  subsets: ["latin"],
  weight: ["400", "700"],
  display: "swap",
});

export const metadata: Metadata = {
  title: "Continua",
  description:
    "Transform conflicts into complementarity. Understand personality as fluid coordinates across six dimensions.",
};

export default function RootLayout({
  children,
}: Readonly<{
  children: React.ReactNode;
}>) {
  return (
    <html lang="en" className={inter.className}>
      <body>{children}</body>
    </html>
  );
}
```

Key details:
- Import Inter from `next/font/google` with weights 400 and 700 per style guide. `display: "swap"` prevents invisible text during load.
- Apply `inter.className` to `<html>` element so all descendant text inherits Inter.
- Import `./globals.css` to activate Tailwind and theme tokens.
- Keep layout as Server Component (no 'use client').
- Metadata provides a meaningful title and description (FOUN-02 requirement).
